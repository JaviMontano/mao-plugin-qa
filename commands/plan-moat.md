# /pqa:plan-moat

Plan the MOAT (Materials, Ontology, Assets, Templates) asset allocation for a plugin. Runs the `plan-moat-strategy` skill directly.

## Usage

```
/pqa:plan-moat [architecture-plan-path]
```

- `architecture-plan-path` -- Path to an architecture plan generated by `/pqa:plan`. If omitted, searches the current directory for an `architecture-plan.md` file.

## Output

- MOAT strategy document with complexity scores, depth allocation, asset inventory, and naming conventions.

## When to Use

- After `/pqa:plan` if you only want to re-run the MOAT strategy (e.g., after changing the architecture plan).
- When adding new skills to an existing plugin and need to determine their MOAT depth.
