# PQA Brand Orchestration

> JM Labs branding for PQA outputs.

## Brand Identity

- **Brand**: JM Labs
- **Product**: Plugin QA (PQA)
- **Tagline**: "The plugin that audits plugins."
- **License**: MIT

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| **Indigo** | `#6366F1` | Primary, headings, table headers |
| **Cyan** | `#22D3EE` | Accent, highlights, links |
| **Dark** | `#1F2833` | Body text |
| **Light** | `#F8F9FC` | Background |

## Typography

| Use | Font |
|-----|------|
| Headings | Inter (system fallback) |
| Body | System default |
| Code | Monospace |

## Output Conventions

- Reports use markdown-excellence format
- Grade badges: A (green), B (blue), C (yellow), D (orange), F (red)
- Footer: `*PQA v3.0 -- JM Labs - Javier Montano.*`
- Artifacts saved as files, never inline

## Prohibited

- Never use Sofka branding (orange `#FF7E08`, gold `#FFD700`)
- Never use MetodologIA branding (navy `#122562`, Poppins font)
- PQA is JM Labs brand exclusively
